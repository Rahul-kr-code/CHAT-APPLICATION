##Real-Time Chat Application (Socket.IO)
📌 Project Overview

This project is a real-time chat application built using Node.js and Socket.IO. It allows multiple users to communicate instantly through a web interface. Messages sent by one user are immediately visible to all connected users without refreshing the page.

This project demonstrates the implementation of WebSocket-based communication and real-time data synchronization between client and server.

🚀 Features
Real-time messaging between multiple users
Instant message broadcasting
No page reload required
Simple and clean user interface
Lightweight and fast communication
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Node.js
Express.js
Socket.IO
⚙️ How It Works
The client connects to the server using Socket.IO
When a user sends a message, it is emitted to the server
The server broadcasts the message to all connected clients
All users receive and display the message instantly
